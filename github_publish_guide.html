📱 GitHub Pages で売上管理システムを公開

🎯 最終目標

iPhoneで動作する売上管理システムを 無料 で公開し、どこからでもアクセスできるようにします。

1

🚀 GitHubアカウント準備

  1. GitHub.com にアクセス
  2. アカウントがない場合は 「Sign up」 で新規登録
  3. 既にある場合は 「Sign in」 でログイン
💡 ポイント: GitHubアカウントは無料で作成できます
2

📂 新しいリポジトリを作成

  1. 画面右上の 「+」 ボタンをクリック
  2. 「New repository」 を選択
  3. 以下を入力：
    • Repository name: sales-management
    • Description: 売上管理システム
    • ⚠️ 必ず「Public」を選択
  4. Create repository をクリック
⚠️ 重要: 「Private」を選択すると公開できません
3

📤 ファイルをアップロード

  1. 「uploading an existing file」 をクリック
  2. 以下のいずれかの方法でファイルを追加：
    • 方法A: index.html ファイルをドラッグ&ドロップ
    • 方法B: 「choose your files」 で選択
  3. ファイルが追加されたら下にスクロール
  4. Commit changes をクリック
📁 使用ファイル: /Users/ko/index.html
4

🌐 GitHub Pages を有効化

  1. リポジトリのページで 「Settings」 タブをクリック
  2. 左メニューの 「Pages」 をクリック
  3. Source で 「Deploy from a branch」 を選択
  4. Branch で 「main」 を選択
  5. Save をクリック
⏱️ 処理時間: 設定後、数分で公開されます
5

🎉 公開URL取得

  1. 数分後にページを更新
  2. 上部に緑色のチェックマーク ✅ と公開URLが表示される
  3. URLをコピー
https://yourusername.github.io/sales-management/
📱 このURLをiPhoneのSafariで開くと売上管理システムが動作します

🎊 完了！

公開されたURLをiPhoneでアクセスしてください

動作確認後、必要に応じてチェックボックスなどの微調整を行います

6

📱 iPhone での使用方法

  1. 公開URLをiPhoneのSafariで開く
  2. 正常に動作することを確認
  3. 「共有」→「ホーム画面に追加」でアプリ風に使用
  4. ブックマークに追加して簡単アクセス

❓ 困ったときは

  • 🔄 ページが表示されない → 数分待ってから再度アクセス
  • 🚫 エラーが出る → Publicに設定されているか確認
  • 📱 iPhoneで動作しない → URLが正しいか確認
